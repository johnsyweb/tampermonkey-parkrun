parkrun Future Roster Printable View acts on parkrun Future Roster pages (e.g. [Logan River Future Roster](https://www.parkrun.com.au/loganriver/futureroster/)), to make them easier to print by removing everything but the table.

It also allows you to edit the contents to clarify when there are multiple roles with the same name, a special event, or if there is text that would be helpful below the table. Simply tap on the text you want to edit.
