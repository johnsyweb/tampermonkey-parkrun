parkrun Future Roster Printable View acts on parkrun Future Roster pages (e.g. [Logan River Future Roster](https://www.parkrun.com.au/loganriver/futureroster/)). The page loads as usual; click **🖨️ Prepare for printing** above the roster when you want a simplified view for editing and printing.

It removes everything but the table, allows you to edit cells to clarify when there are multiple roles with the same name or a special event, and adds an editable note below the table. Reload the page to return to the normal parkrun layout.
